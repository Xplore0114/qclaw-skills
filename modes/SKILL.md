# 🎯 模式切换系统

## 可用模式

| 模式 | 触发词 | 入口 Skill | 核心能力 |
|---|---|---|---|
| 🎙️ 面试模式 | "进入面试模式" | `interview-agent` | 技术拷打、八股问答、模拟面试 |
| 🔬 科研模式 | "进入科研模式" | `research-lab` | PubMed检索、假设驱动、证据追踪 |
| 🏠 正常模式 | "退出模式"、"回到正常模式" | — | 默认全功能 |

## 使用方式

### 进入模式
用户只需说：
- "进入面试模式" → 加载面试技能
- "进入科研模式" → 加载科研技能
- "有哪些模式" → 显示本列表

### 模式内操作
每个模式有独立的能力集，进入后自动加载对应 Skill 的全部功能。

### 退出模式
- "退出模式"
- "回到正常模式"
- 直接问其他无关问题（自动退出）

## 技术实现
- 每个模式 = 一个 Skill（skills/ 目录下的 SKILL.md）
- 模式入口 = SKILL.md 中的 description 触发词
- 无状态切换，进入模式只加载上下文，不锁定会话
