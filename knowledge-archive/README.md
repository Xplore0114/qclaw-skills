# 知识链接即时存档

自动将用户发送的知识类链接（论文、项目、文章、视频）读取、本地存档、飞书同步。

## 功能

- 读取微信文章、GitHub 仓库、arXiv 论文等知识类 URL
- 本地存档到 `memory/knowledge/`
- 有长期价值的同步到飞书 Agent 资料库
- 给用户返回 3-5 句摘要

## 使用方式

用户发送知识类链接时自动触发，无需手动调用。

## 触发条件

- 微信文章：`mp.weixin.qq.com/s/...`
- GitHub 仓库：`github.com/...`
- 论文：`arxiv.org/abs/...`
- 其他技术博客、文档、教程

## 流程

1. 检查 xray 代理是否运行
2. 根据链接类型读取内容
3. 提取标题、核心观点、关键要点
4. 本地存档
5. 飞书同步（可选）
6. 摘要汇报给用户
