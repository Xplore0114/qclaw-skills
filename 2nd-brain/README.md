# 第二大脑

个人知识库，用于记录和检索人物、地点、餐厅、游戏、技术、事件、媒体、想法、组织等信息。

## 触发方式

- 用户提到某个人、地点、餐厅、游戏、设备、事件、书籍/节目、想法或公司
- 触发词：remember、note that、met this person、visited、played、what do I know about 等

## 记录内容

- 人物：名字、关系、互动记录
- 地点：位置、去过的时间、评价
- 餐厅：店名、位置、菜系、体验
- 游戏：游戏名、平台、玩后感
- 技术：工具名、用途、用法
- 事件：事件名、时间、内容
- 媒体：书/影视名、评价
- 想法：公司/人/地点/任何想法

## 检索优先级

命名实体 > 日志
