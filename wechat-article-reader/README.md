# 微信公众号文章读取

通过代理读取微信公众号文章内容，绕过反爬限制。

## 适用场景

用户发送微信文章链接时自动触发。

## 前置要求

- xray 代理运行中
- 使用桌面 Chrome UA
- 代理节点正常

## 流程

1. 检查代理状态
2. 使用代理 + 桌面 UA 请求微信文章
3. 解析 HTML 提取正文
4. 返回文章标题 + 核心内容 + 存档

## 常见问题

| 症状 | 原因 | 解决方案 |
|------|------|----------|
| "环境异常" | 代理未运行或节点失效 | 重启 xray 或换节点 |
| 内容为空 | UA 被识别 | 确保桌面 Chrome UA |
