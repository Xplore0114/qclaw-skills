# 论文投稿模板库

## 一、Cover Letter 模板

### 模板1：首次投稿（英文期刊通用）

```
Dear Editors,

We would like to submit the enclosed manuscript entitled "[论文标题]" for possible publication in [期刊名称].

This paper presents [1-2句话概括研究内容]. Our main findings include: (1) [发现1]; (2) [发现2]; (3) [发现3]. These results demonstrate [研究意义].

This work is novel because [说明创新点]. It is closely related to the journal's scope focusing on [期刊领域], particularly your recent special issue on [相关专题].

The manuscript has not been submitted to any other journals, and all authors have approved the submission.

Thank you for your consideration.

Best regards,
[姓名]
[通讯作者]
[单位]
[邮箱]
[电话]
```

### 模板2：修改稿回复（Response to Reviewers）

```
Dear Dr./Prof. [编辑姓名],

On behalf of all co-authors, we thank you very much for giving us an opportunity to revise our manuscript. We have carefully addressed all the reviewers' comments. Below is our point-by-point response.

[编号] Comment: [审稿人意见]
Response: [我们的回复，包括具体修改内容]

We hope the revised manuscript is now acceptable for publication in [期刊名称].

Thank you again for your time and constructive feedback.

Best regards,
[通讯作者姓名]
```

## 二、审稿意见回复模板

### 结构框架

```
尊敬的编辑和审稿人：

非常感谢您在百忙中审阅我们的稿件并提出宝贵的修改意见。我们已仔细研读每条意见并对论文进行了相应修改。以下是逐条回复：

[审稿人1意见]
1. [意见1]
   回复：...
2. [意见2]
   回复：...

[审稿人2意见]
...

再次感谢审稿人的专业意见，我们已在修改稿中用[红色/高亮]标注了相关修改。

此致
敬礼
[通讯作者]
[日期]
```

### 常见回复句式

| 场景 | 回复句式 |
|------|---------|
| 感谢审稿 | "非常感谢您提出的建设性意见，这有助于提升论文质量。" |
| 同意修改 | "我们完全同意您的建议，已在第X页第X行进行相应修改。" |
| 部分同意 | "感谢您的建议，关于第X点，我们认为[说明理由]，已在文中增加了[说明]。" |
| 解释说明 | "您的理解非常准确，这是由于[原因]，我们已在图1中补充说明。" |
| 文献补充 | "感谢您的建议，我们已添加以下参考文献：[文献信息]。" |
| 语言润色 | "我们已请专业英语编辑对全文进行语言润色。" |

### 应对不同意见

**1. 新颖性不足**
```
回复：感谢您的意见。我们已在第2节增加了对现有方法的综述，明确指出了本文与已有工作的三点不同：(1) ... (2) ... (3) ...
```

**2. 实验不足**
```
回复：感谢您的建议。我们已在第3.2节补充了[实验描述]，结果显示[结论]。实验条件见下表。
```

**3. 文献缺失**
```
回复：感谢您的提醒。我们已添加以下参考文献：
[1] ...
[2] ...
这些工作与本文方法在[方面]有所不同。
```

**4. 格式问题**
```
回复：已按要求修改格式，具体修改包括：(1) 标题字体调整为... (2) 参考文献格式统一为...
```

```

[逐条列出审稿意见和回复]

We hope the revised manuscript meets the requirements for publication in [期刊名称].

Best regards,
[通讯作者]
```

## 五、注意事项

1. **称呼**：尽量查找编辑姓名，写 Dear Dr. [姓]，如无法确认再用 Dear Editor
2. **长度**：Cover Letter 控制在 200-400 字
3. **创新点**：用 3-5 条要点说明研究的新颖性
4. **期刊适配性**：说明论文为何适合该期刊
5. **回复审稿**：逐条回复，标注修改位置（页码/行号）
6. **态度**：始终保持尊重、专业，不争论、不抱怨
