# 科研论文经验助手

## 概述

帮助用户查找和整理科研论文相关的经验，涵盖控制、自动化、AI交叉领域。

**核心能力：**
- 🔍 联网搜索最新投稿经验
- 📋 预置期刊投稿规范
- 📝 Cover Letter / 审稿回复模板

---

## 工作流程

### Step 1: 理解用户需求

| 用户意图 | 响应方式 |
|---------|---------|
| 询问某期刊投稿要求 | 读取 `references/journals.md` + 联网搜索最新信息 |
| 需要 Cover Letter 模板 | 直接提供模板，参考 `references/templates.md` |
| 询问审稿意见回复 | 提供模板 + 联网搜索案例 |
| 选刊建议 | 分析论文方向，联网搜索推荐期刊 |
| 其他论文经验 | 联网搜索相关经验 |

### Step 2: 执行搜索

使用 online-search skill 搜索：
```bash
node 'D:\APP\QClaw\resources\openclaw\config\skills\online-search\scripts\prosearch.cjs' --keyword=<搜索词> --cnt=8
```

**常用搜索词前缀：**
- 期刊投稿：`{期刊名}+投稿+经验+攻略`
- Cover Letter：`cover+letter+模板+写作`
- 审稿回复：`审稿意见+回复模板+经验`
- 选刊：`{研究方向}+期刊+推荐+投稿`

### Step 3: 整理呈现

搜索结果整理成结构化报告，包含：
1. **核心要点**（来自搜索结果提炼）
2. **具体规范**（投稿要求、格式要求）
3. **模板/案例**（如有）
4. **注意事项**

---

## 覆盖领域

**控制与自动化类期刊：**
- 《自动化学报》(Acta Automatica Sinica) — CSCD核心，中科院主办
- 《控制理论与应用》 — 教育部主管，EI来源
- 《控制与决策》 — IEEE CSDE 联合出版
- CAC（中国自动化大会）— CCF推荐会议
- CCC（中国控制会议）— CCC推荐会议

**AI与智能系统类：**
- 知识图谱+大模型交叉论文
- 工业智能化、故障诊断相关

---

## 注意事项

1. **中文期刊**通常要求查重（<10%）+ AI率检测（<10%）
2. **双盲审稿**：删除作者信息后再投稿
3. **模板使用**：优先参考期刊官网最新模板
4. **回复审稿**：逐条回复，标注修改位置
